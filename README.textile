h1. Badger

This is a data storage project build for the company I work at, it's fully open source if you care for using it and should be easy to setup

h2. Status
!https://travis-ci.org/nohm/badger.png?branch=master!:https://travis-ci.org/nohm/badger
!https://gemnasium.com/nohm/badger.png(Dependency Status)!:https://gemnasium.com/nohm/badger

h2. Requirements

* Ruby 2.0.0 or newer
* Rails 4.0.0 or newer

h2. Database

* SQLite with ActiveRecord

h2. Development

* Template Engine: ERB
* Testing Framework: Test::Unit (not implemented yet)
* Front-end Framework: Twitter Bootstrap (Sass) & Bootswatch
* Form Builder: Bootstrap Form
* Authentication: Devise
* Authorization: CanCan
